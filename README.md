# 💼 Job Tracker – Personal Job Application Manager

🌐 Live Demo: https://web-job-tracker.netlify.app/dashboard

---

## 📌 Overview

**Job Tracker** is a web application designed to help users manage and organize their job application process efficiently.

It provides a centralized dashboard to track applications, monitor progress, and stay organized throughout the job search journey.

---

## 🎯 Problem Statement

Job seekers often apply to multiple companies and struggle to track:

* Application status
* Deadlines
* Interview stages
* Follow-ups

This leads to missed opportunities and poor organization.

**Job Tracker solves this by providing a structured system to manage and visualize job applications.**
